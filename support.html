快速开始

  1. 访问 api.walletexplain.com 注册账号
  2. 充值余额或购买订阅套餐
  3. 进入 API 密钥 → 创建密钥，分组选择 default-pool
  4. 复制 API Key（只显示一次，请妥善保存）
项目值
API 地址（Base URL）https://api.walletexplain.com/v1
认证方式Authorization: Bearer 你的API_KEY
计费单位美元（USD），按 Token 用量扣费

curl 测试

将 你的API_KEY 替换为控制台创建的密钥：

curl https://api.walletexplain.com/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer 你的API_KEY" \
  -d '{
    "model": "gpt-4o-mini",
    "messages": [{"role": "user", "content": "你好"}]
  }'

成功时返回 JSON，其中 choices[0].message.content 为 AI 回复内容。

LobeHub 接入（推荐）

  1. 打开 lobechat.com，用 Google / GitHub 登录
  2. 进入 设置 → 语言模型 → OpenAI

  4. 选择模型 gpt-5.4 或 gpt-4o-mini，点击 检查

常见问题

401 Invalid API Key

检查 Key 是否复制完整；确认 Key 未过期、未被删除；确认使用的是用户账号下的 Key，而非误用其他账号。

INSUFFICIENT_BALANCE（余额不足）

账户余额不足，请在控制台充值，或联系客服。

Encrypted content is not supported with this model

请改用 gpt-5.4，或联系管理员将上游账号的 Responses API 设为「强制 Chat Completions」。

网站打不开 / 连接超时

检查 DNS 是否解析正确；Mac 用户可将 DNS 设为 8.8.8.8 和 223.5.5.5。在终端执行 nslookup api.walletexplain.com，应显示 47.237.31.107。

API Key 和 OpenAI 官网 Key 有什么区别？

Sub2API 生成的 Key 用于访问 api.walletexplain.com，从你的账户余额扣费。OpenAI 官网 Key 是上游密钥，由平台配置，用户无需也不应直接使用。
