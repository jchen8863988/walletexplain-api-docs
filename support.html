产品介绍

WalletExplain API 是一个 OpenAI 兼容的 AI 模型网关。注册并充值后，在控制台创建 API Key，即可通过标准接口调用 GPT 等模型，适用于 LobeHub、curl、Python / Node SDK 等开发场景。

  • 接口兼容 OpenAI API，接入成本低
  • 按 Token 用量计费，余额以美元（USD）显示
  • 支持流式输出（stream: true）
  • 国内可访问，无需自行配置 OpenAI 官方账号

快速开始

  1. 访问 api.walletexplain.com 注册账号
  2. 充值余额（或使用兑换码）
  3. 进入 API 密钥 → 创建密钥，分组选择 default-pool
  4. 复制 API Key（只显示一次，请妥善保存）
  5. 在客户端填入下方 API 地址和你的 Key，开始调用

curl 测试

将 你的API_KEY 替换为控制台创建的密钥：

普通请求

curl https://api.walletexplain.com/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer 你的API_KEY" \
  -d '{
    "model": "gpt-4o-mini",
    "messages": [{"role": "user", "content": "你好"}]
  }'

成功时返回 JSON，其中 choices[0].message.content 为 AI 回复内容。

流式请求

curl https://api.walletexplain.com/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer 你的API_KEY" \
  -d '{
    "model": "gpt-4o-mini",
    "stream": true,
    "messages": [{"role": "user", "content": "你好"}]
  }'

常见问题

API Key 和 OpenAI 官网 Key 有什么区别？

请在 WalletExplain API 控制台 API 密钥 中创建 Key，并配合地址 https://api.walletexplain.com 使用；调用费用从你的 WalletExplain 账户余额 扣除。

不要使用 OpenAI 官网（platform.openai.com）的 Key。OpenAI 官网 Key 无法在本服务中使用，也不能替代 WalletExplain API Key。

在 LobeHub 中：API Key 填 WalletExplain 的 Key，API 代理地址填 https://api.walletexplain.com（不要填 api.openai.com）。

401 Invalid API Key

检查 Key 是否复制完整；确认 Key 未被删除；确认使用的是当前登录账号下创建的 Key，而非其他账号的 Key。

INSUFFICIENT_BALANCE（余额不足）

账户余额不足，请登录控制台充值，或使用兑换码，或联系客服。

Encrypted content is not supported with this model

请优先改用 gpt-5.4。若仍报错，请联系客服处理。

网站打不开 / 连接超时

检查 DNS 是否解析正确。Mac 用户可将 DNS 设为 8.8.8.8 和 223.5.5.5。在终端执行 nslookup api.walletexplain.com，应显示 47.237.31.107。

ChatBox 显示 Load failed

多为客户端本地网络或代理问题。请先用 curl 或 LobeHub 验证 API 是否正常，再排查 ChatBox 的网络设置。

如何查看用量和余额？

登录 api.walletexplain.com，在控制台首页或用量记录中查看当前余额与 Token 消耗。
